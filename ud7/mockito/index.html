    3. Dobles de prova: Mockito

    Dobles de prova amb Mockito

    Mockito és una biblioteca popular de Java utilitzada per crear dobles de prova (mocks) de manera senzilla i eficient. Els dobles de prova, s'utilitzen en les proves per aïllar el codi que es vol provar de les dependències externes (com bases de dades, serveis web, altres components del sistema, etc.).

    Què és un mock en el context de les proves?

    Un mock és un tipus de doble de prova que substitueix una dependència real per una versió controlada, creada explícitament per comprovar que les interaccions entre el codi que estem provant i les seves dependències es produeixin de manera correcta.

    Els mocks són útils quan volem verificar que certes mètodes es criden amb els arguments correctes, o quan volem evitar que certes dependències reals (com connexions de xarxa o de base de dades) facen l'execució de les proves més lenta o complexa.

    Com funciona Mockito?

    Mockito permet crear mocks de classes o interfícies, establir el comportament esperat dels mètodes (per exemple, què han de retornar quan se'ls cride) i verificar les interaccions després de l'execució de la prova.

    Aplicació a la calculadora

    Veiem com creariem una doble prova per a la calculadora.

    Abans que res, haurem d'incorporar la llibreria Mockito al nostre projecte.

    Per a això, al fitxer app/build.gralde afegim a les dependències:

    repositories {
    // Use Maven Central for resolving dependencies.
    mavenCentral()
}

dependencies {
  ...
    // https://mvnrepository.com/artifact/org.mockito/mockito-core
    testImplementation("org.mockito:mockito-core:5.17.0")
}

    Una vegada guardat el fixer, haurem de sincronitzar el projecte amb Gradle.

    Una vegada sincronitzat, creem la classe CalculadoraTestService:

    package com.ieseljust.edd;

import static org.mockito.Mockito.*;
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.assertEquals;

public class CalculadoraServiceTest {

    @Test
    public void testService() {
        Calculadora mockCalculadora = mock(Calculadora.class);
        when(mockCalculadora.sumar(3, 4)).thenReturn(7);

        assertEquals(7, mockCalculadora.sumar(3, 4));
    }
}
